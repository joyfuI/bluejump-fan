# 블루점프 팬 사이트

[블루점프](https://www.bluejump.net/ko/) 팬 사이트입니다.

## 기능

* 라이브 모아보기: 현재 방송 중인 분들을 확인할 수 있습니다. 우측 하단에 멀티뷰로 열기 기능이 있습니다.
* 캘린더 모아보기: 숲 방송국 캘린더의 내용을 가져오는 거라 스트리머가 캘린더 기능을 사용하지 않는다면 뜨지 않습니다. 우측 위 드롭다운 메뉴로 과거 캘린더도 확인할 수 있습니다.
* 방송국글 모아보기: 방송국 게시판 글 목록을 불러옵니다. 방송국 별로 첫 번째 페이지만 불러옵니다. 우측 하단의 필터 기능으로 원하는 스트리머 글만 표시할 수 있습니다.
* 카페글 모아보기: 팬카페의 스트리머 전용 게시판 글 목록을 불러옵니다. 카페 게시판 별로 첫 번째 페이지만 불러옵니다. 우측 하단의 필터 기능으로 원하는 스트리머 글만 표시할 수 있습니다.
* 다시보기 모아보기: 다시보기들을 한 번에 모아서 보여줍니다. 방송국 별로 첫 번째 페이지만 불러옵니다. 우측 하단의 필터 기능으로 원하는 스트리머 다시보기만 표시할 수 있습니다.

방송 도구
* 댓글 업순 따잇쇼: 숲 댓글 실시간 업 랭킹을 보여줍니다. 게시글 주소를 입력하면 되며, 하이라이트 ID를 지정하면 해당 ID를 스크롤로 따라가며 보여줍니다.

## 개발 환경 설정

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm run dev
```

개발 서버가 실행되면 [http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

## 빌드 및 배포

```bash
# 프로덕션 빌드
pnpm run build

# 빌드 서버 실행
pnpm run preview
```
